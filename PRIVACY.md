# プライバシーポリシー
「にゃんとく」(以下「本アプリ」) は、利用者の個人情報を尊重し、適切に取り扱うため、以下のとおりプライバシーポリシーを定めます。

**最終更新日**: 2026年4月29日

---

## 1. 取得する情報
本アプリは、サービスの提供のために以下の情報を取得します。

### 1.1 アカウント情報
- **匿名認証 ID**: 初回起動時に Firebase Authentication が自動生成する識別子。氏名・住所・電話番号など個人を特定する情報は含まれません。
- **ニックネーム**: 利用者が任意に登録するひらがな表記の名前。
- **登録年月**: 利用者が任意に入力する登録時の年月情報。

### 1.2 アプリ内データ
- 育成した猫の情報（種類、名前、好感度、ステータス等）
- 奉納履歴・コレクション情報
- ルーム参加情報、およびルーム内での活動履歴

### 1.3 端末情報
- **プッシュ通知トークン (FCM Token)**: プッシュ通知の配信のため。
- OS 種別、アプリバージョン等の技術情報。

本アプリは、Firebase Analytics、Crashlytics 等のアクセス解析・クラッシュレポート機能は利用していません。広告識別子 (IDFA / GAID) も取得しません。

---

## 2. 利用目的

取得した情報は、以下の目的のためにのみ利用します。
- 本アプリの基本機能（猫の育成、奉納、ルーム共有等）の提供
- 利用者間の共有体験の実現（共有棚への奉納反映等）
- プッシュ通知の配信
- 不正利用の防止およびサービス品質の維持・改善
- 利用者からのお問い合わせへの対応

---

## 3. 第三者への提供

本アプリは、利用者の同意がある場合、または法令に基づく場合を除き、取得した情報を第三者に提供しません。
ただし、ルーム機能等の共有体験においては、ニックネーム、奉納した猫の情報など、サービス上で公開を前提とする情報が他の利用者から閲覧可能となります。

---

## 4. 利用する外部サービス

本アプリは、サービスの提供のために以下の外部サービスを利用しています。
| サービス名 | 提供事業者 | 利用目的 |
|---|---|---|
| Firebase Authentication | Google LLC | 利用者認証 |
| Cloud Firestore | Google LLC | アプリデータの保存・同期 |
| Cloud Functions for Firebase | Google LLC | サーバサイド処理 |
| Firebase Cloud Messaging | Google LLC | プッシュ通知の配信 |

データは Google Cloud のアジア東京リージョン (asia-northeast1) に保管されます。Google における処理の詳細は [Google プライバシーポリシー](https://policies.google.com/privacy) をご確認ください。

---

## 5. データの保管期間
利用者が本アプリを継続して利用する間、データを保管します。アカウント削除をご希望の場合は、後述のお問い合わせ先までご連絡ください。削除依頼に基づき、合理的な期間内にデータを削除します。

---

## 6. データの安全管理
本アプリは、取得した情報の漏洩、改ざん、滅失等を防止するため、以下を含む適切な安全管理措置を講じます。

- 通信経路の暗号化 (HTTPS / TLS)
- Firestore セキュリティルールによるアクセス制御
- 認証された利用者のみがデータにアクセスできる設計
- サーバ側処理 (Cloud Functions) における権限チェック

---

## 7. 利用者の権利
利用者は、本アプリで取得された自身の情報について、以下の権利を有します。

- 開示請求
- 訂正・削除請求
- 利用停止請求

これらの請求は、後述のお問い合わせ先までご連絡ください。本人確認のうえ、合理的な期間内に対応します。
